# Expo client — pinned to SDK 56

This app is pinned to **Expo SDK 56** (React Native 0.85, React 19.2,
TypeScript 6.0.3) per the architecture stack pin. Do not bump the SDK as part of
an unrelated change — a version bump is an architecture decision (correct-course),
not a drive-by.

Read the exact versioned docs for the pinned SDK before writing code:
https://docs.expo.dev/versions/v56.0.0/

Keep native/SDK dependencies aligned with `expo install` / `expo install --fix`
rather than editing versions by hand.
